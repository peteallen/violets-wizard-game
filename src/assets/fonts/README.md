# Andika font files

The game self-hosts the Latin subsets of Andika Regular and Bold so early-reader captions render consistently without a network dependency. The files are from the Andika v27 release distributed by Google Fonts, and the accompanying `OFL.txt` is the SIL Open Font License supplied with that family.
