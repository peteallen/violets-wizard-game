# Bundled game fonts

The game self-hosts the Latin subsets of Andika Regular and Bold so early-reader captions render consistently without a network dependency. The files are from the Andika v27 release distributed by Google Fonts, and the accompanying `OFL.txt` is the SIL Open Font License supplied with that family.

Source URLs recorded on July 12, 2026:

`https://fonts.gstatic.com/s/andika/v27/mem_Ya6iyW-LwqgwarYQeL8WVQ.woff2`

`https://fonts.gstatic.com/s/andika/v27/mem8Ya6iyW-Lwqg40ZMFVZ0bf8pkAg.woff2`

The title uses the Latin subset of Almendra Bold as its decorative storybook face. Almendra is reserved for display lettering; Andika remains the readable interface face. The font is from the Almendra v28 release distributed by Google Fonts, and `OFL-Almendra.txt` records its SIL Open Font License.
