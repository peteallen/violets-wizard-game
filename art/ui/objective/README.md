# Objective reminder prop

`source/objective-reminder-v2-chroma.png` is the untouched built-in
image-generation source for Violet's compact objective reminder. The painting
contains only the parchment, leather, stitching, and brass holder. The runtime
draws the detailed compass, short objective caption, close seal, and every
state-dependent layer.

The exact prompt, built-in generation receipt, source and shipping hashes, and
deterministic chroma-removal/WebP processing live in `../../field-kit-v1.*` and
`../../process-field-kit-v1.mjs`.
