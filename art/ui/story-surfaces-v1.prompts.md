# Violet's Field Kit story surfaces v1 production prompts

All assets in this set are textless transparent material holders. Runtime code
continues to own characters, portraits, icons, labels, colors, selection,
progress, focus, and input state.

Every source was generated on a perfectly flat solid `#00ff00` chroma-key
background with no background shadow, gradient, texture, reflection, floor, or
lighting variation. The key color does not appear in the painted subject.

## Choice tag

Use case: stylized-concept

Asset type: transparent-background reusable dialogue-choice prop

Create one richly hand-painted near-square portrait-choice card, slightly wider
than tall: an irregular warm parchment sheet mounted on a shallow
blackberry-leather backing with plum stitching, rubbed edges, two tiny
aged-brass corner fasteners, and a soft hand-shaped silhouette. Keep the upper
two-thirds calm and open for a live character portrait or vector emblem. Keep
the bottom quarter quiet and blank for a live one-to-three-word caption without
making it look like a nested button.

silhouette and generous padding. No text, character, animal, portrait, face,
icon, emblem, wax seal, check, star, glow, selection mark, status badge, button
