# Violet's Field Kit DOM and browser identity v1 production prompts

The dialog folio is a textless nine-slice material shell. Native DOM inputs,
labels, copy, validation, focus, actions, busy state, and save data remain live.
The browser icon is a stable opaque identity mark intended for browser tabs and
iPad home-screen masking; it contains no live game state.

## Dialog folio

Use case: stylized-concept

Asset type: reusable nine-slice game UI dialog folio for Violet's Field Kit

Primary request: Create one richly hand-painted square storybook document folio
that can frame accessible HTML dialogs for pet naming and save transfer.

Scene/backdrop: perfectly flat solid `#00ff00` chroma-key background for local
background removal.

Subject: one front-facing blackberry-purple leather document holder with softly
irregular hand-cut edges, plum stitching, four modest aged-brass corner
protectors, and a large uninterrupted warm cream parchment center. The object
should read as a premium school travel folio made for a young witch, matching
the tactile quality of an illustrated satchel: visible worn leather grain,
paper fibers, tiny scuffs, and warm upper-left painted light.

Style/medium: richly detailed hand-painted children's storybook game UI prop,
tactile gouache and digital painting, magical but practical, no software chrome.

Composition/framing: square 1:1 artboard; the complete folio centered and
filling about 88 percent of the frame; front-on with negligible perspective.
Keep the outer 18 percent border as the stable leather/brass frame and the inner
64 percent as calm parchment so the image can be used as a CSS nine-slice with
fill. Corners and edge thickness must be visually consistent.

Color palette: blackberry leather, muted plum stitching, honey parchment, aged
brass, warm brown shadows.

Materials/textures: worn pebbled leather, hand stitching, fibrous parchment,
scuffed brass.

Constraints: exactly one folio; no text, letters, numbers, logos, watermark,
owl, character, portrait, wax seal, X, button, tab, bookmark, input field,
textarea, icon, star, glow, state badge, cast shadow, contact shadow, floor
plane, reflection, or action-like ornament. The entire center must remain quiet
and readable for live DOM content. Background must be one uniform `#00ff00`
with no shadows, gradients, texture, reflections, floor plane, or lighting
variation. Keep crisp separated edges and generous padding. Do not use
`#00ff00` anywhere in the folio.

Avoid: software panel, perfect vector geometry, ornate central decoration,
nested frame inside the parchment, pure black, pure white.

Receipt: `exec-28185515-6872-4ccc-8acd-9d95e8fe2813`

## Owl clasp browser icon

Use case: stylized-concept

Asset type: browser favicon and iPad home-screen icon for Violet's Wizard Game

Primary request: Create one original, richly hand-painted owl clasp emblem for
a magical children's storybook game, designed to remain immediately
recognizable at 32 pixels.

Scene/backdrop: full-bleed deep blackberry-purple pebbled leather field, edge
to edge, with restrained warm upper-left painted light.

Subject: one centered front-facing aged-brass owl face clasp with a very strong
simple silhouette: two large warm honey enamel eyes, a small heart-shaped brow
bridge, compact beak, and only the minimum engraved feather marks needed to
read as an owl. The owl should feel friendly, curious, and handcrafted for
Violet's school satchel, not fierce or heraldic.

Style/medium: premium hand-painted storybook game icon, tactile gouache and
digital painting, high material detail at source but simplified high-contrast
value grouping for tiny-size legibility.

Composition/framing: square 1:1 icon; centered symmetrical owl clasp occupying
about 68 percent of width; generous safe margin for iOS masking; no explicit
rounded-square border and no clipped corners.

Color palette: deep blackberry leather, aged honey brass, cream and amber eyes,
warm brown recesses.

Materials/textures: worn pebbled leather, scuffed brass, tiny enamel inlays.

Constraints: exactly one owl clasp; no text, letters, numbers, logo wordmark,
watermark, stars, wand, compass, satchel silhouette, wax seal, button, glow,
sparkles, cast shadow, contact shadow, floor plane, border, corner radius,
mockup, device frame, or transparent background. Full square must be opaque.
Original design only.

Avoid: photoreal animal, angry eyes, skull-like face, excessive filigree, thin
lines that vanish at small size, pure black, pure white, neon colors.

Receipt: `exec-7fa0fb28-a17b-486c-85ed-641fc47eded0`
