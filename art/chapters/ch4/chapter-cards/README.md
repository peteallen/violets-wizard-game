# Chapter Four chapter-card previews

The flying-lesson preview was generated on July 17, 2026 with Codex's built-in image-generation workflow. The initial prompt used `art/raw/ch2-lake-vista-v2.png` for the project's original distant castle and blue-violet magical atmosphere, and `art/raw/ch1-platform.png` for the polished storybook finish, dimensional light, and luminous-gold accent language. The built-in workflow returned no model identifier, provider route, seed, or billable-usage record, so none is claimed.

The untouched initial source, `art/raw/ch4-flying-lesson-native-v1.png`, was not safe to ship because it contained five golden star rings instead of the requested three or four. Its exact prompt, request, rejection, and hashes remain in `flying-lesson-native-v1.prompt.md`, `flying-lesson-native-v1.request.json`, and `flying-lesson-native-v1.metadata.json`. A single tightly scoped built-in edit removed only the smallest highest ring. That correction is documented by `flying-lesson-native-v2.prompt.md`, `flying-lesson-native-v2.request.json`, and `flying-lesson-native-v2.metadata.json`.

The accepted untouched corrected source is `art/raw/ch4-flying-lesson-native-v2.png`; the accepted 2048×1152 PNG is `art/raw/ch4-flying-lesson-v2.png`; and the shipping derivative is `public/assets/art/chapter-cards/ch4-flying-lesson.webp`. The native source was center-cropped by one pixel with `sips -s format png -c 940 1672`, resized once with `sips -s format png -z 1152 2048`, and encoded with `cwebp -q 88 -m 6 -sharp_yuv`.

Source inspection accepted v2 with one riderless broom, exactly four golden star rings, the original distant castle, a safe autumn lawn, no people, creatures, or readable text, and calm upper and lower runtime zones. Shipping inspection at 2048×1152 confirmed that the crop and encode preserve the single broom, four-ring path, castle continuity, title and action space, and painted texture without a visible conversion defect. The runtime supplies all title and control text separately.
