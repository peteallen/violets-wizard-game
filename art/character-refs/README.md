# Painted character references

These images are design targets for the deterministic Canvas puppets. They are committed under `art/` for side-by-side review and prompt provenance, but they are not runtime sprites: nothing in this directory belongs in `public/assets` or the production asset manifest.

Every canonical sheet uses the same landscape construction: one front-facing full-body view on the left and a more detailed face close-up on the right, both depicting the same identity. The room paintings remain the fidelity standard. A sheet is accepted only when it passes the Storybook Standard for palette, line, form, the reference-sheet’s upper-left key, texture, organic shape, and the screenshot squint test. Runtime puppets mirror that reference lighting when a room’s authored key arrives from the opposite side.

The shared style anchors are `../raw/ch1-malkins.png`, `../raw/ch1-menagerie.png`, and `../raw/ch1-diagon-dusk.png`; each character also uses the room most closely associated with that character. Exact prompts and returned metadata live beside each accepted PNG.
